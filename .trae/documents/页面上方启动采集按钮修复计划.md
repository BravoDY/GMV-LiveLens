# 页面上方「启动采集」按钮修复计划

## 一、现象

正式环境页面 (`/`) 顶部「启动采集」按钮 (`#schedulerToggle`) 点击无反应。

## 二、根因分析

### 证据链

1. **`#schedulerToggle` 按钮在 HTML 中存在且可见** — [index.html:L45](file:///c:/Users/yjd22/Desktop/python项目/GMV-LiveLens/frontend/index.html#L45)：
   ```html
   <button id="schedulerToggle" class="scheduler-button" type="button">启动采集</button>
   ```
   正式环境 `/` 路径不调用 `applyPublicDashboardMode()`，按钮不会被隐藏。

2. **但整个 frontend/ 目录中没有任何代码为 `#schedulerToggle` 绑定 click 事件** — `core.js:L751-757` 全局事件绑定区注册了 `testOcr`、`saveTask`、`captureAllButton`、`previewRemotePage`、`resumeAfterLogin`，**唯独没有 `schedulerToggle`**。

3. **后端调度器 API 已就绪** — [system.py:L80-L96](file:///c:/Users/yjd22/Desktop/python项目/GMV-LiveLens/backend/routers/system.py#L80-L96)：
   - `GET /api/scheduler` — 获取调度状态（`{running: true/false, ...}`）
   - `POST /api/scheduler/start` — 启动采集
   - `POST /api/scheduler/pause` — 暂停采集

4. **已有 CSS 样式就绪** — `.scheduler-button` / `.scheduler-button.active` 在 [styles.css:L495-L503](file:///c:/Users/yjd22/Desktop/python项目/GMV-LiveLens/frontend/styles.css#L495-L503)。

### 根因结论

**`#schedulerToggle` 从未被绑定 click 事件**。按钮本身就缺了一个 JS 按钮处理逻辑。

## 三、修复方案

在三个地方修改：

### 3.1 在 `core.js` 全局事件绑定区新增 `schedulerToggle` 点击逻辑

**文件**: [core.js](file:///c:/Users/yjd22/Desktop/python项目/GMV-LiveLens/frontend/core.js)，在 L756 之后新增函数 `toggleScheduler()` + 事件绑定：

```javascript
async function toggleScheduler() {
  const btn = $("schedulerToggle");
  if (!btn) return;
  btn.disabled = true;
  try {
    const status = await api("/api/scheduler");
    if (status.running) {
      await api("/api/scheduler/pause", { method: "POST" });
      btn.textContent = "启动采集";
      btn.classList.remove("active");
    } else {
      await api("/api/scheduler/start", { method: "POST" });
      btn.textContent = "停止采集";
      btn.classList.add("active");
    }
  } catch (e) {
    showMessage(parseApiError(e) || "调度器操作失败", true);
  } finally {
    btn.disabled = false;
  }
}
```

并注册事件：`$("schedulerToggle")?.addEventListener("click", () => toggleScheduler());`

### 3.2 在 `app.js` 中加调度器状态同步

在 `startInternalDashboard()` 末尾调用 `syncSchedulerButton()` 从 API 读取调度器运行状态并设置按钮文字和样式。

### 3.3 更新 version 号

`index.html` 中 `core.js`、`app.js` 版本号更新。
